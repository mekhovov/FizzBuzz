fizz_buzz
=========

program printing numbers from 1 to 100 but:
*   "FIZZ" instead of multiples of 3
*   "BUZZ" instead of multiples of 5
*   "FIZZ BUZZ" instead of multiples of 3 and 5


Hot to run:
---------
*   just call

        ruby main.rb
